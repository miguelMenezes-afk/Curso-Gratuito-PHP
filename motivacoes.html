// motivações

Por que criamos
esse projeto?

Este projeto nasceu dentro da faculdade de Engenharia de Software com um propósito claro: tornar o aprendizado de programação acessível a todos, independentemente de condição financeira ou localização geográfica.

PHP é uma das linguagens mais utilizadas no mundo — está por trás de gigantes como WordPress, Facebook e Wikipedia. Dominar PHP abre portas para o mercado de trabalho e para a construção de sistemas web completos.

Acreditamos que conhecimento é um bem público. Por isso, todo o conteúdo deste curso é e sempre será gratuito.

🌎

Democratizar o acesso

Remover barreiras financeiras para que qualquer pessoa possa aprender programação de qualidade.

💼

Impulsionar carreiras

PHP é a base de 78% dos sites com back-end conhecido. Aprender PHP abre portas reais no mercado.

🎓

Extensão curricular

Colocamos em prática nossa formação para gerar impacto social real na comunidade.

🤝

Comunidade ativa

Criamos um espaço colaborativo onde alunos aprendem juntos e se apoiam mutuamente.
